## 🔧 技术实现细节

### 底栏标题修改技术架构

```
用户输入 → NSUserDefaults 存储 → DYYYBottomTabTitleMapping()
    ↓             ↓                        ↓
(设置界面)  (配置键: DYYYModifyBottomTabText)  (解析映射字典)
                                              ↓
                                    layoutSubviews 应用
                                              ↓
                                      setTitle/setAccessibilityLabel
                                              ↓
                                           生效
```

### 主题系统技术架构

```
用户配置 → NSUserDefaults 存储 → DYYYGetThemeXXX()
   ↓            ↓                      ↓
(设置) (多个配置键) (获取 UIColor)
                      ↓
              DYYYUtils.colorFromSchemeHexString()
                      ↓
                   应用颜色
                      ↓
                    生效
```

### 关键技术点

1. **Hook 系统**
   - 使用 Logos 框架 Hook 系统类
   - `%hook AWENormalModeTabBar` 对底栏进行定制

2. **数据持久化**
   - NSUserDefaults 存储所有配置
   - 支持字典序列化

3. **动态应用**
   - layoutSubviews 中实时应用修改
   - 支持运行时动态更新

4. **颜色处理**
   - 复用现有的 DYYYUtils 工具类
   - 支持多种颜色格式
